Description

This is an example of a full page preferences editor. As preferences are changed, they are visible in a preview window. After the preferences are saved, they are applied to the entire page.

Instructions

  1. Experiment with the various Preferences Editor settings and verify that the preview window contents are styled correctly.
  2. Use the "save and apply" button to apply the preview styles to Preferences Editor
  3. Use the "reset" button to reset the styles
