Description

This is an example of a separated panel preferences editor. As preferences are changed, they will be visible in the main content area. After the panel is closed, the settings will be applied to the preferences editor. This particular example is intended to showcase the preferences editor working with a variety of content.

Instructions

  1. activate the "Show Display Preferences" button in the top left corner of the screen
  2. experiment with the various Preferences Editor settings and verify that the window contents are styled correctly.
