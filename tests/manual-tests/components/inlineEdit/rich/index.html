Description

Below are three examples of rich text inline edit. Two using TinyMCE, and one using CK Editor 3. Apart from being a demostration of the rich text inline edit itself, this also demostrates that mulitple instances can be run on the same page.

Note: TinyMCE and CK Editor are pulled down from a CDN, which may prevent some of their features from working (i.e. anything using a dialog).

Instructions

  1. Activate the the Edit link
  2. Modify the content in the rich text editor
  3. Activate the save button to accept the changes
  4. Activate the undo/redo link to toggle between undoing and redoing the changes
